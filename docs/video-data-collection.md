# 📱 视频数据收集模板

## 提交格式

每个视频请按以下格式提供：

```
【城市】杭州/上海/北京/深圳/广州/成都
【分类】学区房/新房/二手房/投资/租房/政策
【抖音分享链接】https://v.douyin.com/xxxxx/
【播放量/点赞】从链接前的数字提取，如：5.64（表示564万播放）
【文案】完整的视频口播内容（这最重要，用于AI洗稿）
```

## 目标

- **数量**：50-100个视频
- **分布**：每个城市10-20个
- **类型**：覆盖学区房、新房、二手房、投资等
- **优先级**：播放量 >100万的爆款视频

## 注意事项

1. **文案必须完整**：AI洗稿依赖原始文案，请尽量提供完整口播内容
2. **分类准确**：方便用户按需求查找
3. **真实数据**：播放量和点赞数影响热词排序
